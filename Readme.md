## Project Summary 

This project is a personal portfolio website designed to showcase my skills, projects, and background as a front-end web developer.
It features a modern UI, a transparent blurred header, a hero section with background image, smooth hover animations, and responsive sections for:
About Me
Skills
Projects
Contact Form
Animated footer scene
The website is built as a one-page portfolio, keeping navigation clean and minimal.
